Facial Recognition

This Python script uses scikit-learn to implement facial recognition via principle component analysis.

  • Identified faces by Principle Component Analysis through using various machine learning libraries in Python such as scikit learn
  • Achieved maximum precision of 85% using only profiles facing straight forward or up
  • With all profiles used Principle component analysis was less successful, as the visual cues were too inconsistent.
